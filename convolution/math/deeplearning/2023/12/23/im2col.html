What is convolution?

Convolutional is a mathematical operation which could take inpus as two-dimensional matrices to four-dimensional matrices and the convolutional filters can also range from two-dimensional matrices to four-dimensional matrices. The above equation gives a unified view of this, where x is the input and k is the convolutional kernel giving us a as the result of the convolution which is often called a feature map

In the above picture x represents the input, w represents the convolutional filter and A represents the feature map. The process of convolution is to divide the input x into several smaller matrices of same size as the convolutional kernel and then do elementwise multiply and sum with the convolutional kernel.

Now we can express each of the smaller matrices into a row vector like the following example

Doing the same over for all of the smaller matrices would yield us a new matrix which we will call X

In the same way conolutional filter could be expanded as a column vector and we will call it W

Now convolution is just a matrix multiplication between X and W
